Why Memory Search?

🧠

Semantic Understanding

Finds content by meaning, not just keywords. Searching "authentication decisions" finds JWT discussions even if the word "authentication" isn't used.

⚡

Lightning Fast

Model caching means subsequent queries in under 0.1ms. Indexing runs at ~150 documents per second. You'll spend less time waiting and more time finding.

🔒

Private & Local

No data leaves your machine. Embeddings run locally with Transformers.js. SQLite backend ensures crash safety and concurrent access.

🗓️

Date Filtering

Search specific time periods: "show me decisions from last month" or "what happened in January 2026" with precise date ranges.

🌐

Multiple Interfaces

CLI for interactive use, HTTP API for integrations, OpenClaw plugin for chat commands. Works however you work.

🔧

Production Ready

SQLite with WAL mode supports concurrent readers/writers. ACID transactions prevent corruption. Battle-tested by real users.
